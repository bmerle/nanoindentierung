# Unsupervised Machine Learning für die Nanoindentierung
Übung zur **Vorlesung Nanoindentierung**
Dieses Projekt zeigt, wie unüberwachtes Lernen (KMeans, GMM) genutzt wird, um Nanoindentierungsdaten zu clustern und die Phasen eines vielphasigen Werkstoffs zu unterscheiden. Ziel ist es, mechanische Eigenschaften automatisch einzelnen Materialphasen zuzuordnen und komplexe Datensätze besser zu interpretieren.

Universität Kassel, 2026
